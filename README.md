# Choix Difficile

Une petite application pour aider à la prise de décision lorsque plusieurs options s'offrent à nous.
Fondamentalement, c'est une sélection aléatoire parmi les choix proposés.

## Installation

Commencez par installer les dépendances :

```bash
make install
```

Le répertoire `src/` contient tous les fichiers de l'application, vous pouvez donc le déployer tel quel.

### Environnement de développement

Pour installer le programme en mode développement, utilisez plutôt cette commande :

```bash
make install-dev
```

Cela installe JQuery dans son mode "développement", et permet ainsi de profiter de ses fonctions de débogage.
